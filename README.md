# The Daily Crumb Bakery Project
A fully responsive, multi-page web application featuring custom branding, real Kenyan pricing structures, accessible layouts, and interactive theme options.

## Features Built
- 7 Interlinked HTML5/Bootstrap 5 Pages
- Custom Espresso-Cream Typographic Stylesheet
- JavaScript Dynamic Light/Dark Theme Controller
- Client-side Form Validation Scripting
